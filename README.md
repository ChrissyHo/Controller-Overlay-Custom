# PS5 DualSense Controller Overlay Pack

A fully customizable **PS5 DualSense controller overlay** designed for OBS, streaming, and web display.  
This package includes animated overlays, SVG layers, PNG assets, and ready-to-use HTML pages.

Perfect for:
- Streamers who want a clean input display  
- Custom controller themes  
- Animated overlays (WEBM)  
- VSCView users  
- Anyone wanting a transparent PS5 controller silhouette with input highlights

## üéÆ How to Use in OBS

1. Open **OBS Studio**  
2. Add ‚Üí **Browser Source**  
3. Point it to either:
   - A **local file** (e.g., `ps5-controller-overlay.html`)
   - A **hosted GitHub Pages link** (recommended)

4. Set:
   - Width: **1920**
   - Height: **1080**
   - Enable **Transparent Background**

5. (Optional) Add your **animated WEBM** as a media source above the controller image.
